=====================================
Sphinx Themes for Plone Documentation
=====================================

sphinx.themes.plone is a collection of `Sphinx`_ themes for the `Plone`_ documentation project.

The different themes are used on `docs.plone.org`_, but are not limited for this purpose.
It might be used for all Plone Package Documentations, or even private usecase but than without the Plone Logo and Footer (respect the :ref:`license`).

This documentation is divided into three parts:

- Simple Usage description for documenting Plone Core or related Collective Packages (see :doc:`usage`)
- The Theme Flags - how to configure the Theme appreance in depth (see :doc:`advanced`)
- Further Development of sphinx.themes.plone (see :doc:`development`)
